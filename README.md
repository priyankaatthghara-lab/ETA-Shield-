# ETA Shield 🚢

**Real-time Intelligence for Resilient Supply Chain Management**

ETA Shield calculates resilient delivery windows and downstream impacts by analyzing shipment variables and disruption events in real-time.

## 🎯 Features

- **Live Shipment Tracking** – Monitor current position
- **Disruption Analysis** – Vessel delays, storms, and other event impact prediction
- **Severity Index** – Visual severity scale (Minor → Catastrophic) for quick risk assessment
- **Cargo Prioritization** – High-priority cargo flagging (e.g., Semiconductors)
- **Predictive ETA** – Original ETA comparison with disruption-adjusted delivery windows

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Icons:** Remix Icon
- **Data:** Real-time shipment API integration

## 📦 Use Case

Supply chain managers can input shipment variables (vessel ID, cargo type, disruption scope) to instantly calculate adjusted delivery timelines and assess downstream impacts on dependent operations.

## 🚀 Installation

```bash
git clone https://github.com/yourusername/eta-shield.git
cd eta-shield
# Open index.html in browser (no build step required)
```

## 🧑‍💻 Usage

1. Enter current shipment position
2. Input original ETA timestamp
3. Select disruption type (Vessel Delay / Storm)
4. Specify cargo criticality
5. View adjusted severity index and delivery window
